#百度之星初赛A All X
> F(x,m) 代表一个全是由数字x组成的m位数字。请计算，以下式子是否成立：<br>
> F(x,m) mod k ≡ c <br>
> 1≤ x ≤9 <br>
> 1≤ m ≤1010 <br>
> 0 ≤ c < k≤10,000 <br>

* 找寻环节，设 g(i) :表示 i 为全由 x 组成的数 mod k 后的余数；<br>
* 很明显，g(i) = ( g(i-1)\*10 + x ) mod k ，同余模定理；<br>
* 很明显，g(i)的循环节大小 <= k ;<br>
* 那我么就暴力处理出循环节即可,要注意一下，这里并不是纯循环的；<br>
